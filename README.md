# Task 3: Interactive Prototype Creation

## Project Title

IRCTC Website Redesign - Interactive Prototype

## Objective

The objective of this task was to transform a static website redesign into an interactive prototype that simulates real user interactions and navigation flows.

## Tool Used

Canva

## Project Description

An interactive prototype of the IRCTC website was created based on the redesigned user interface developed in Task 2. The prototype allows users to navigate between different sections of the website through clickable elements and page transitions.

## Screens Included

1. Home Page
2. Book Ticket Page
3. PNR Status Page
4. Help Page

## Features Implemented

* Clickable navigation menu
* Interactive page transitions
* User-friendly navigation flow
* Consistent visual design across all screens
* Home navigation available on all pages

## Prototype Flow

Home → Book Ticket

Home → PNR Status

Home → Help

Book Ticket → Home

PNR Status → Home

Help → Home

## Outcome

The interactive prototype successfully demonstrates the user journey and core website interactions. The final design provides a modern, user-friendly, and visually appealing experience compared to the original website.

## Conclusion

This task helped in understanding the process of converting static UI designs into interactive prototypes, testing usability, collecting feedback, and refining the user experience.
